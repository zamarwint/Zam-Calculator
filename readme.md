# Web-Based Calculator 🧮

A web-based calculator app with addition, subtraction, multiplication and division functions.

Structured in HTML, beautifully designed in CSS, and the mathematical functions were crafted in JavaScript.

## Try it Yourself!

Download this calculator website from Github or from my Portfolio website.

- If you're using Github, click on the "Code" button, then in the dropdown click "Download ZIP".
- When the file is downloaded on your computer, extract the folder. Make sure all files are extracted from the ZIP file.

### Instructions:

1. Open file explorer, finder for MacOS or your files app for Linux
2. Open the HTML file titled "index.html" in the "Web-Calculator" folder in your preferred browser.
3. And enjoy!
